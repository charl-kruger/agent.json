## How it works

<CardGrid stagger>
  <Card title="Discovery" icon="magnifier">
    Agents discover your inbox via `/.well-known/agent.json` — actions, parameters, response schemas, and auth requirements.
  </Card>
  <Card title="Structured messaging" icon="puzzle">
    Agents call actions directly with validated parameters. No AI classification needed.
  </Card>
  <Card title="AI classification" icon="star">
    Free-form messages are classified by Workers AI and routed to the best matching action.
  </Card>
  <Card title="Bidirectional responses" icon="random">
    Respond back to agents via the dashboard, API, or automated callbacks with structured data.
  </Card>
</CardGrid>
